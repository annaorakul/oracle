Услуги и стоимость
Расклад на месяц (Ленорман) 22 BYN +

Прогноз по событиям в отношениях, работе, финансах на ближайший месяц. Можно с любого дня месяца на 30 дней. Нужны: имя, дата рождения, фото.

Диагностика отношений (Ленорман / Таро) 32 BYN +

Анализ текущей ситуации, прогноз / перспектива (в зависимости от расклада).

Для тех, кто только начинает отношения, либо в паре возникли сложности, либо вы оказались в любовном треугольнике. Также для тех, у кого ещё нет отношений, но хочется это изменить.

Нужны: имена, даты рождения, фото (Ленорман); имена (по желанию — дата рождения, фото) для Таро.

Финансовый разбор (нумерология) 55 BYN +

Где самые большие деньги, что ограничивает приход финансов, что нужно делать, чтобы привлечь финансы, денежные сферы.

Нужны: дата рождения.

Прогноз на год (нумерология) 66 BYN +

Прогноз по месяцам, ключевые тенденции по сферам; практические наполняющие действия, хобби, цвет года и цвета месяца (и где их применять).

Нужны: дата рождения.

Прогноз на год (астрология) 75 BYN +

Планирование по периодам, события года, задачи и возможности, поддерживающие практики.

Нужны: дата, время и место рождения.

Прогноз на год (Ленорман) + гармонизация 85 BYN +

Основные события по сферам (отношения, работа, финансы), дополнительная информация индивидуально по ситуации + ритуал на смягчение событий.

Нужны: имя, дата рождения, фото.
